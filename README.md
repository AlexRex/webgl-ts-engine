# WebGL 3D motor engine

Pure WebGL 3D motor engine built with Typescript.

## Getting Started

### Prerequisites

You must have NPM installed.

### Installing

Install dependencies

```
$ npm install
```

## Running

```
$ npm run start
```

## Build

```
$ npm run build
```
